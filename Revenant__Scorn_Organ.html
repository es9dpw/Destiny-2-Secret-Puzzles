Destiny 2 Secret Puzzles

Scorn Organ

Information

The scorn organ is found in the Kell's Fall exotic mission, at the far end of the room that opens after fighting the trickster for the first time

To enter melodies into the scorn organ you must stand on the plate in the middle and theh shoot the right notes in order, before stepping off the plate to confirm the melody

Each melody in this guide will be described with numbers, 1-13, from left to right, as shown in the image below:

Melody 1

1, 8, 1, 9

This code seemling does nothing of use, but is still required for the scorn organist triumph

Melody 2

2, 11, 1, 10, 7, 6

This code unlocks the door for the 2nd intrinsic upgrade for Slayer's Fang. This opened door can be found by following the stairs to the right of the organ

Melody 3

1, 4, 11, 1, 4, 10

This code unlocks the door for the 3rd intrinsic upgrade for Slayer's Fang. This opened door can be found straight ahead after leaving the room hwere you first enter a mirror.

Melody 4

1, 8, 1 ,6

This code unlocks the door for the 4th intrinsic upgrade for slayer's fang. This opened door can be found in the room with the rally flag, before the mindbender fight, but you must activate 3 lanterns to open the door first.

Melody 5

1, 8, 4, 11, 1, 6, 4, 10

This code open a small room during the Kell's Fall mission whihc has a shrine to prince uldren, althought it doesn't have any practical purpose, and it is still required for the scorn organist triumph

Melody 6

1, 8, 1, 11, 9

This code seemling does nothing of use, but is still required for the scorn organist triumph

Melody 7

4, 7, 9, 6, 2, 6, 7, 1

This code plays a voice line of Savathun laughing, but it doesn't have any practical purpose, and it is still required for the scorn organist triumph

Melody 8

10, 4, 6, 1, 11, 4, 8, 1

This code will teleport you into a boss fight, that is required to complete the key fieldwork for act 3 of Episode Revenant.
